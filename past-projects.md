# Android Automotive Infotainment Development - System and Apps

**Timeline:** Since 2024 (Launch planned for 2025)

## Key Deliverables
- **System Applications (Vendor Apps):**
  - Vehicle Settings: Exterior/interior configurations (lights, mirrors, driving modes)
  - Driver Assistance Settings: ADAS customization (lane-keeping, adaptive cruise, parking assist)
  - HVAC Control: Temperature, fan speed, seat heating/cooling
  - Connectivity & Bluetooth: Device pairing and network management
  - OTA Updates: Firmware management

- **System UI Components:**
  - Core infotainment UI elements (Top Bar, Bottom Bar, Navigation, Dialogs, Themes)

- **Emulator:**
  - Custom app configuration for testing bypass signals
  - UI-first validation and expanded mocked signals for testing
